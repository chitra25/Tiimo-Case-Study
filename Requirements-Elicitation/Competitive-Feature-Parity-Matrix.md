# Competitive Feature Parity Matrix : Tiimo iPhone Churn Reduction Initiative
**Phase 1 Elicitation Artifact**  
**Sources**: App Store, Product Websites, User Reviews 

---

## Executive Summary
This competitive feature matrix benchmarks Tiimo's current Routines functionality against four primary ADHD focused planning competitors: Structured, Llama Life, TickTick, and Sunsama. Analysis reveals that Tiimo's removal of the Routines feature library creates a significant competitive gap, as all benchmarked competitors offer saved routine templates with timed subtasks. This gap directly correlates with the 22% of VOC complaints citing Routines removal and validates the strategic priority of the Routines MVP restoration.

**Key Finding**: Routines are table stakes in the ADHD planner market. Tiimo's absence of this feature represents a critical competitive disadvantage driving user migration to alternatives.
